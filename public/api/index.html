NEWS API

This is the root of the api for our news website - here are the major endpoints.

  • /topics
  • /users
  • /articles

Topics

A list of topics with a description and an image url.

Users

A list of users with properties including username, name, an avatar url and a favourite topic (the topic they have most frequently commented on)

Articles

A list of articles with various properties including a count of all comments. Articles can be patched to increase the votes by sending a javascript object with a key of "inc_votes" and a positive or negative number to increment or decrement the votes

Articles by id

/articles/:id - using the article's numerical id to get a single article

Article comments

/articles/:id/comments - a list of all the comments in a single article. Posting a comment requires a JavaScript object with a username (must be the username of an existing user in the database) and a body.

Optional queries for Articles

  • sort_by - the column to sort the table by (defaults to created_at)
  • order - "asc" or "desc" for ascending or descending sort order (defaults to descending)
  • topics - a topic to filter the articles by
